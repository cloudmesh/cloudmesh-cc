class Queues(filename=None, database=None)[source]

Bases: object

Queues holds all of the queues with their corresponding names. It is a meta-queue, essentially. The queues class will be a dictionary of dictionaries of jobs, which are job names and commands.

An example command would likely look like:

cms cc queues list –queues=ab

Initializes the giant queue structure.
