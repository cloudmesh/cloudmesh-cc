HTTP Routing Table

/run
GET /run/{name} Run Workflow
 
/upload
POST /upload Upload
 
/workflow
GET /workflow/{name} Get Workflow
DELETE /workflow/{name} Delete Workflow
 
/workflows
GET /workflows List Workflows
